Sistema de Cadastro de Reservas

O projeto atual é um protótipo para criação de um aplicativo de cadastro de reservas de quartos para alunos de uma faculdade. O aplicativo deve possuir o nome do aluno, matrícula, data, perído e verificação de cadastro.
Também foi adicionada a opção de escolher o tipo de quarto.
